Funny Oscilloscope Terminal

This is the code for I used to make an arduino display up to 64 (legibly) or 256 (illegibly) characters on an Oscilloscope for CMU's 2024 Build18 Hackathon.
Arduinos are probably worse than you may realize, and so this task involved several hacks and optimizations to get a working end result of displaying my laptop's terminal on the scope.
As I don't have time right now, I won't detail the whole process here yet, but for anyone interested, the code for what I did exists.
